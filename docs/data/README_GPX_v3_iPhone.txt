Ducati Scrambler Alps Road Book - GPX v3

This pack contains routepoint GPX files, one per riding day.

Important:
- These GPX files intentionally do NOT include a straight-line track.
- Import each day as a ROUTE into OsmAnd, Kurviger, Calimoto, Garmin, TomTom, etc.
- Let your navigation app calculate the real roads between the curated routepoints.
- This avoids the fake accuracy problem where a GPX track draws straight lines between towns/passes.
- For iPhone offline use, OsmAnd or Organic Maps with pre-downloaded France/Switzerland/Benelux maps is safer than relying on Google Drive preview.

Fuel planning:
- Planned fuel top-ups roughly every 180-200 km, conservative for a Ducati Scrambler with luggage and Alpine riding.

Safety:
- Do not interact with the road book while riding. Use it stopped, or use voice navigation from your navigation app.
