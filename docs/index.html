Ducati Alps Road Book

Installable iPhone road-book app for the Ducati Scrambler Alps trip. It includes offline schematic maps, GPX routepoints, big bike-friendly controls, and cached files for low-signal areas after the first load.

Install on iPhone / bike-mounted use

PWA mode: open this GitHub Pages link in Safari, then use Share → Add to Home Screen. The app will launch full-screen with its own icon and cache the road book files.
Bike use: use the road book as a planning/checkpoint tool when stopped. For moving navigation, open the route in Google Maps or import GPX into OsmAnd, Kurviger, Calimoto, Garmin or TomTom with voice guidance.
1
Add to Home ScreenOpen the GitHub Pages URL in Safari → Share → Add to Home Screen → launch it like an app.
2
Pre-download mapsUse Google Maps offline areas or OsmAnd/Organic Maps offline maps for France, Switzerland, Belgium, Luxembourg and the Netherlands.
3
Use GPX in a navigation appFor the bike, import GPX v3 into OsmAnd, Kurviger, Calimoto, Garmin or TomTom and let that app calculate the roads.

GPX v3 accuracy notes

The GPX pack has been rebuilt to be safer for motorcycle navigation. I removed the misleading straight-line tracks and kept ordered routepoints + waypoints. This means your navigation app should calculate the actual road route between the points.

🧭
Use route calculationImport each day as a route, not as a fixed straight-line track.
⛽
Fuel points includedDaily routepoints include planned fuel options around the conservative 180–200 km strategy.
🏍️
Bike useUse this HTML as the road book; use OsmAnd/Kurviger/Calimoto for moving navigation.

Made for an iPhone mounted on a Ducati Scrambler: large buttons, offline maps, conservative riding time, nature camping, easy gravel only where legal/open.
